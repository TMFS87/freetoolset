📱 QR Code Generator

Generate QR codes from any text or URL. Customize colors, size, and error correction. Download as a PNG image instantly.

🔒 All processing happens in your browser. No data is uploaded to any server.

📋 How to Use

  1. Enter the text or URL you want to encode in the input box.
  2. Adjust the QR code size using the slider (128px – 512px).
  3. Customize the dark and background colors if desired.
  4. Select an error correction level (higher = more resilient).
  5. Click "Generate" and download the QR code as a PNG image.

❓ FAQ

What content can I encode in a QR code?

You can encode any text, including URLs, plain text, phone numbers, email addresses, WiFi credentials, and more. QR codes can hold up to 2,953 bytes of data.

How large can I generate the QR code?

You can generate QR codes from 128px to 512px in size. For print use, we recommend generating at 512px and scaling up in your design software to maintain crispness.

Does this tool require an internet connection?

The QR code generation happens entirely in your browser using JavaScript. Once the page is loaded, no internet connection is needed. The only external resource is the QR code library loaded from a CDN.

What is error correction level?

Error correction allows a QR code to be read even if part of it is damaged or obscured. Higher levels (like H/30%) are more resilient but produce denser codes. Use High for codes that may be printed or worn.
