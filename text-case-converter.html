❓ FAQ

What is Title Case?

Title Case capitalizes the first letter of every word in the text. For example, "hello world" becomes "Hello World". It is commonly used for book titles, article headlines, and section headers. Some style guides exclude small words (articles, conjunctions, prepositions) from capitalization, but this tool applies a simple rule: capitalize the first letter of every word.

What is the difference between camelCase and snake_case?

camelCase capitalizes the first letter of each word except the first, and removes all spaces — e.g. "hello world" becomes "helloWorld". It is commonly used in JavaScript, Java, and C# variable names. snake_case uses underscores between words and keeps everything lowercase — e.g. "hello world" becomes "hello_world". It is commonly used in Python, Ruby, and database column names.

How do I convert to Sentence case?

Sentence case capitalizes only the first letter of each sentence and leaves the rest lowercase. Click the "Sentence case" button to convert your text. The tool detects sentence boundaries using periods, exclamation marks, and question marks, then capitalizes the first letter following each boundary.

What is kebab-case used for?

kebab-case uses hyphens between words in all lowercase — e.g. "hello world" becomes "hello-world". It is commonly used in URLs (slugs), CSS class names, and file names, as hyphens are URL-safe and more readable than underscores in web addresses.
