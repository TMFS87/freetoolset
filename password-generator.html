🔐 Password Generator

Generate strong, random passwords with customizable length and character sets. Uses the Web Crypto API for maximum security.

🔒 All processing happens in your browser. No data is uploaded to any server.

📋 How to Use

  1. Set your desired password length using the slider (4–64 characters).
  2. Select which character types to include (upper, lower, numbers, symbols).
  3. Optionally enable "Exclude ambiguous characters" for readability.
  4. Click "Generate Password" to create a new secure password.
  5. Copy the password using the "Copy" button.

❓ FAQ

Are these passwords secure?

Yes. This tool uses the Web Crypto API's crypto.getRandomValues() function, which generates cryptographically secure random numbers. This is the same standard used by banks and security applications.

How long can I make the password?

You can generate passwords from 4 to 64 characters. We recommend at least 16 characters for strong security. Longer passwords are exponentially harder to crack.

What characters are supported?

You can include uppercase letters (A-Z), lowercase letters (a-z), numbers (0-9), and symbols (!@#$%^&*()_+-=[]{}|;:,.<>?). You can also exclude visually ambiguous characters like I, l, 1, O, and 0.

How does the strength indicator work?

Strength is calculated based on password length and character variety. A password with 16+ characters using all four character types (upper, lower, numbers, symbols) will be rated "Strong".
