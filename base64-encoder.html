🔄 Base64 Encoder / Decoder

Encode text to Base64 or decode Base64 back to readable text. Supports UTF-8 characters including Chinese, Japanese, and emoji.

🔒 All processing happens in your browser. No data is uploaded to any server.

📋 How to Use

  1. Select the mode: "Encode" (text to Base64) or "Decode" (Base64 to text).
  2. Type or paste your text into the input box.
  3. The result updates automatically as you type (live mode).
  4. Use "Swap" to move the result to the input and switch modes.
  5. Click "Copy Result" to copy the output to your clipboard.

❓ FAQ

Can it handle Chinese and other Unicode characters?

Yes. The tool uses proper UTF-8 encoding and decoding, so it correctly handles Chinese, Japanese, Korean, emoji, and any other Unicode characters that standard Base64 implementations may struggle with.

Can I encode files to Base64?

Currently, the tool processes text input only. For file-to-Base64 conversion (e.g., creating Data URIs), you can use the browser's File API in combination with this tool. A dedicated file encoding feature is planned.

Is there a size limit?

The limit depends on your browser's available memory. In practice, you can process text up to several megabytes without issues. Very large inputs may cause the browser to become temporarily unresponsive.

What is Base64 used for?

Base64 is commonly used to encode binary data as text, enabling it to be safely transmitted over text-based protocols like email, HTTP, and JSON. It's also used in Data URIs, JWT tokens, and Basic HTTP authentication.
